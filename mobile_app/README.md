# Car Rental Mobile App (Flutter MVP)

This folder contains a standalone Flutter mobile app for the Django backend in this repository.

## MVP Scope
- Login
- Dashboard
- Vehicles list + details
- Rentals list + details
- Create rental form

## Run
```bash
cd mobile_app
flutter pub get
flutter run
```

## API Status
Current backend has no production REST APIs yet. The app is wired to a minimal planned API contract documented in `docs/backend_api_plan.md`.
